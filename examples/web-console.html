pm2-web-logger HTML Example

Single-page SSE console for PM2 logs.

Fill in the server URL, stream name, and stream filter. The page loads the latest buffered lines and then keeps the console live through SSE.

Connection

This page uses fetch() to consume SSE, so it also works when AUTH_TOKEN is enabled. If you open this HTML from another origin, make sure CORS is configured on the server. When this page runs over https://, insecure API URLs are upgraded to https:// to avoid mixed content.
